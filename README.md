## Base 全家桶编解码

### 基于base64模块编写

#### 用法
···
git clone https://github.com/iamjazz/pythonbase.git
cd pythonbase
···

```
python3 Decode.py  
请输入你想转码的编码：
1.Base64解码;
2.Base85解码;
3.Base16解码;
4.Base32解码;
请输入编号：1
请输入要解码的Base64编码:
```

```
python3 Encode.py  
请输入你想转码的编码：
1.Base64解码;
2.Base85解码;
3.Base16解码;
4.Base32解码;
请输入编号：1
请输入要编码的Base64编码:
```
